📌 Project Overview

This project presents an end-to-end data analysis solution for Zomato sales data, transforming raw datasets into actionable business insights using Power BI.

It focuses on:

- 📊 Customer behavior
- 🌍 City-level performance
- 🍽️ Product trends

---

🎯 Business Objective

- Identify key revenue drivers
- Analyze top-performing cities & categories
- Understand user engagement patterns
- Support data-driven decision making
